# Web Search

## Overview

The familiar should be able to search the web to answer questions about current events, look up facts, fetch patch notes, etc.

## Things to Research

- What search APIs are available and suitable
- How to safely inject web content into the familiar's context without enabling prompt injection attacks
- Whether a summarization intermediary (smaller model) between raw search results and the familiar is the right approach
- How to give the owner control over what can and can't be searched
